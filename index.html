Intro

Hello, my name is Florebencia Fils-Aime. I am a recent Computer Engineering graduate from the University at Buffalo (UB). I am passionate about social justice, supporting more women in STEM and inclusion. In my free time, I like to draw pictures and write poetry. Here's a fun fact about me: I used to have 6 fingers on each hand but it was cut off after I was born. I still have the scar on my hand to this day. By the way, check out my awesome work.

Past Projects:

  • OpenPose Sign Language Recognition (February - May 2020) - I worked with Jasleen Alexis on this project. I collected surveys of research papers about Sign Language Recognition. I helped to gather videos and photos to collect hand keypoints from OpenPose. At the end of the semester, we presented the challenges of using OpenPose and the research we collected. You can see the video below.


  • The Sixth Sense (August - December 2019) - I developed an embedded device for deaf and/or partially blind drivers and utilized C++ to create the classes for the device. Here's the code


  • Hermes (August - December 2019) - I worked in a team to create a web application that makes a traveling itinerary based on a budget. My teammates were Danny Delannes-Molka, Javier Falca, Shanelle Ileto, and Sarah Peters. I utilized JavaScript, Google Places API, Google Maps API, and other Google Platform Console services to work on frontend and to collect user data.
  • Hotel Lock Safe- I designed a binary hotel safe using an Atmel 8-bit AVR Microcontroller, resistors, buttons, and 4 LEDS. The user would input a 6-bit binary number using the buttons and then the lock would be in the locked state. It will only unlock if that same 6-bit binary number is used. See the code here.




  • Mobile Game Application- I explored an interest in mobile games during my Freshman and sophmore year. It was a third person shooter game created in Java. See the code here.
  • Self-Love Calendar - I developed an interactive java application my freshman year that used user input to select a day in the calendar, and displayed an inspiring quote based on the day. See the code here.

About

I was born on January 23, 1998 in Montreal, Canada. I was raised in by Haitian parents for my whole life and was raised in both Brooklyn, NY and Queens, NY. I started coding when I was a freshman in University. I started coding because I was interested in Computer Science and I wanted to be a model for black women who want a career in STEM.

Throughout most of my life, I have always been a good student. In middle school, I rose to the top of my class and became the validorian in 2012. With my GPA, I was able to get into the honorable Townsend Harris High School (in Flushing, NY) of which I was able to take honors classes from 2012 to 2016. Within those years, I was able to become the Vice President of my school's Gender-Sexuality Alliance, be a part of yearly school performances and Arista (which is a National Honors Society), and gain the Thomas P. Sweetin Award for Promotion of Human Values and Diversity.

During my time at UB, I have been on the Dean's List three times.
